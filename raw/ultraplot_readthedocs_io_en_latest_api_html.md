All of the below objects are imported into the top-level namespace. Use help(uplt.object) to read the docs during a python session. Please note that UltraPlot removes the associated documentation when functionality is deprecated (see What’s New ). However, UltraPlot adheres to effort versioning , which means old code that uses deprecated functionality will still work and issue warnings rather than errors. Important The documentation for “wrapper” functions like standardize_1d and cmap_changer from UltraPlot &lt; 0.8.0 can now be found under individual PlotAxes methods like plot() and pcolor() . Note that calling help(ax.method) in a python session will show both the UltraPlot documentation and the original matplotlib documentation. Figure class ¶ The figure class used for all ultraplot figures. Figure () The Figure subclass used by ultraplot. Grid classes ¶ The gridspec and subplot grid classes used throughout ultraplot. GridSpec ([nrows, ncols, layout_array, ...]) A GridSpec subclass that permits variable spacing between successive rows and columns and hides &quot;panel slots&quot; from indexing. SubplotGrid ([sequence]) List-like, array-like object used to store subplots returned by subplots . Axes classes ¶ The various axes classes used throughout ultraplot. Axes (*args, **kwargs) The lowest-level Axes subclass used by ultraplot. PlotAxes (*args, **kwargs) The second lowest-level Axes subclass used by ultraplot. CartesianAxes (*args, **kwargs) Axes subclass for plotting in ordinary Cartesian coordinates. PolarAxes (*args, **kwargs) Axes subclass for plotting in polar coordinates. GeoAxes (*args, **kwargs) Axes subclass for plotting in geographic projections. ThreeAxes (*args, **kwargs) Simple mix-in of ultraplot.axes.Axes with Axes3D . ExternalAxesContainer (*args[, ...]) Container axes that wraps an external axes instance. Top-level functions ¶ The starting point for creating ultraplot figures. figure (**kwargs) Create an empty figure. subplot (**kwargs) Return a figure and a single subplot. subplots (*args, **kwargs) Return a figure and an arbitrary grid of subplots. show (*args, **kwargs) Call matplotlib.pyplot.show . close (*args, **kwargs) Call matplotlib.pyplot.close . switch_backend (*args, **kwargs) Call matplotlib.pyplot.switch_backend . ion () Call matplotlib.pyplot.ion . ioff () Call matplotlib.pyplot.ioff . isinteractive () Call matplotlib.pyplot.isinteractive . Configuration tools ¶ Tools for setting up ultraplot and configuring global settings. See the configuration guide for details. Configurator ([local, user, default]) A dictionary-like class for managing matplotlib settings stored in rc_matplotlib and ultraplot settings stored in rc_ultraplot . rc Instance of Configurator . rc_ultraplot A dictionary-like container of ultraplot settings. rc_matplotlib A dictionary-like container of matplotlib settings. use_style (style) Apply the matplotlib style(s) with matplotlib.style.use . config_inline_backend ([fmt]) Set up the ipython inline backend display format and ensure that inline figures always look the same as saved figures. register_cmaps (*args[, user, local, default]) Register named colormaps. register_cycles (*args[, user, local, default]) Register named color cycles. register_colors (*args[, user, local, ...]) Register named colors. register_fonts (*args[, user, local, default]) Register font families. Constructor functions ¶ The constructor functions used to build class instances from simple shorthand arguments. Proj (name[, backend, lon0, lon_0, lat0, ...]) Return a cartopy.crs.Projection or Basemap instance. Locator (locator, *args[, discrete]) Return a Locator instance. Formatter (formatter, *args[, date, index]) Return a Formatter instance. Scale (scale, *args, **kwargs) Return a ScaleBase instance. Colormap (*args[, name, listmode, filemode, ...]) Generate, retrieve, modify, and/or merge instances of PerceptualColormap , ContinuousColormap , and DiscreteColormap . Norm (norm, *args, **kwargs) Return an arbitrary Normalize instance. Cycle (*args[, N, samples, name]) Generate and merge Cycler instances in a variety of ways. Locators and formatters ¶ Various Locator and Formatter classes. IndexLocator ([base, offset]) Format numbers by assigning fixed strings to non-negative indices. DiscreteLocator (locs, **kwargs) A tick locator suitable for discretized colorbars. DegreeLocator (*args, **kwargs) Locate geographic gridlines with degree-minute-second support. LongitudeLocator ([lon0]) Locate longitude gridlines with degree-minute-second support. LatitudeLocator (*args, **kwargs) Locate latitude gridlines with degree-minute-second support. AutoFormatter ([zerotrim, tickrange, ...]) The default formatter used for ultraplot tick labels. SimpleFormatter ([precision, zerotrim, ...]) A general purpose number formatter. IndexFormatter (labels) Format numbers by assigning fixed strings to non-negative indices. SciFormatter ([precision, zerotrim]) Format numbers with scientific notation. SigFigFormatter ([sigfig, zerotrim, base]) Format numbers by retaining the specified number of significant digits. FracFormatter ([symbol, number]) Format numbers as integers or integer fractions. CFDatetimeFormatter (fmt[, calendar, units]) Format dates using cftime.datetime.strftime format strings. AutoCFDatetimeFormatter (locator, calendar[, ...]) Automatic formatter for cftime.datetime data. AutoCFDatetimeLocator ([maxticks, calendar, ...]) Determines tick locations when plotting cftime.datetime data. DegreeFormatter (*args, **kwargs) Formatter for longitude and latitude gridline labels. LongitudeFormatter ([lon0]) Format longitude gridline labels.
